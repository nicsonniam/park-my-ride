# Park My Ride

A Next.js project to help motorcyclists in Singapore find nearby HDB, URA, and private parking spots based on geolocation.  

Built with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app) and optimized using [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) with the [Geist](https://vercel.com/font) font.

Features

- Mobile-friendly search for motorcycle parking
- Integrated HDB & URA parking data
- Crowdsourced private carpark information
- Location-based nearest parking recommendations
